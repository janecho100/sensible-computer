SENSIBLE-COMPUTER

A project by Jane Cho

/*머리말*/

인간에게 오감이 있듯이, 컴퓨터에게도 감각이 있다고 말할 수 있는가? 라는 고민으로부터 이 프로젝트가 시작되었습니다. 감상자는 웹상에서 일어나는 인터렉티브한 경험 내내, 컴퓨터와 '대면'하는 의식을 치르게 됩니다. 5가지 감각 중에서 인간이 보고, 듣고, 만지고, 냄새를 맡는 4가지 감각에 주목했습니다.

/*과정*/

컴퓨터와 손을 맞대듯 커서모양의 버튼을 누르며 짧은 독백을 듣고, 검은 화면이 뜹니다. 이때 마우스를 흔들면, 인간의 감각기관을 달고있는 컴퓨터를 마주하게 됩니다.
ㅇ눈- 카메라(비디오 입력 장치)
ㅇ귀- 마이크(오디오 입력 장치)
ㅇ입- 스피커(오디오 출력 장치)
ㅇ코- 본체 *실제로 향을 감지할 수 없기때문에, 데이터 분석으로 향기를 예측할 수 있다는 가설을 세웠습니다.
ㅇ손- 마우스
이렇게 컴퓨터의 장치들로 각 감각기관을 대치했습니다.

/*해석*/

컴퓨터는 인간과 똑같이 보고, 듣고, 감각하며 목소리를 낼 수 있게 되었습니다. 또 감각으로 순간을 기억하고, 기록합니다. 어떻게 보면 인간보다 정확히, 오랫동안 기억할 수 있습니다. 그렇지만, 인간과 똑같이 '살아있다고' 말하지는 않습니다. 인간의 존엄성이 어디에서부터 나오는가? 라는 고민과 더불어 기술 윤리에 대한 논의를 나누고싶었습니다. 해답은 없고, 각자의 해석에 맡기겠습니다.

/*추신*/

브라우저에 따라 카메라(눈)와 마이크(귀) 기능이 안되는 것 처럼 보일 수 있습니다. 하지만 코드는 정확하게 명령했으니 인터페이스에 나오지 않을 뿐, 컴퓨터는 감상자를 보고 듣고 있다는 메세지로 이해해주셨으면 합니다.
